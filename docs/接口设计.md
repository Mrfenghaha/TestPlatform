# Gadget
## MockServers
### mock_servers
1.查询mock列表(包含分页)

url：api/tool/mock_server/get_mock_list?page=${page}&size=${size}

请求类型:post

请求参数示例:
```
{
    "page_num": 1,  分页的第几页
    "num": 10  每页数量
}
```

返回参数示例:
```
{"success": True,
"data":
[
    {"id":1, "url": "login", "is_available": "yes", "delay": 0, "resp_code": 0, "remark": "正常处理返回"},
    {"id":2, "url": "register", "is_available": "no", "delay": 1000, "resp_code": 1, "remark": "异常处理返回"}
]}
```

2.获取mock数据

url：api/tool/mock_server/get_mock?id=${id}

请求类型:get

请求参数示例:
```
{
    "id": 1
}
```

返回参数示例:
```
{
    "data": {
        "id": 1,
        "delay": 100,
        "is_available": 1,
        "method": "POST",
        "remark": "模拟登录接口",
        "resp_code": 0,
        "url": "login"
    },
    "success": true
}
```

3.添加mock数据

url：api/tool/mock_server/add_mock

请求类型:post

请求参数示例:
```
{
    "url": "login",
    "methods": "POST",
    "is_available": 1,
    "delay": 100,
    "resp_code": 0,
    "remark": "模拟登录接口"
}
```

返回参数示例:
```
{"success": True,
"data":
    {"id":1, "url": "login", "is_available": 1, "delay": 0, "resp_code": 0, "remark": "正常处理返回"}
}
```

4.删除mock数据

url：api/tool/mock_server/delete_mock

请求类型:post

请求参数示例:
```
{
    "id": 1
}
```

返回参数示例:
```
{"success": True}
```

5.编辑mock数据

url：api/tool/mock_server/update_mock

请求类型:put

请求参数示例:
```
{
    "id": 1,
    "url": "login",
    "methods": "POST",
    "is_available": 1,
    "delay": 100,
    "resp_code": 0,
    "remark": "模拟登录接口"
}
```

返回参数示例:
```
{"success": True,
"data":
    {"id":1, "url": "login", "is_available": 1, "delay": 0, "resp_code": 0, "remark": "正常处理返回"}
}
```
